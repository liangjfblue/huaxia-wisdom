# 🏯 华夏老祖宗.skill

**AI只会说"建议你权衡利弊"，老祖宗直接说"别骗自己了"。**

纠结跳槽还是留下，AI让你"列优缺点清单"——**老子**：反者道之动，不纠结的时候自然就清楚了。

项目太复杂无从下手，AI说"拆成子任务"——**庖丁**：你硬砍，能不累吗？先看纹理。

老板画饼你该怎么办，AI说"保持积极心态"——**韩非子**：你手里有牌，只是没看清楚。

道理都懂就是做不到，AI说"相信你自己"——**王阳明**：知而不行，只是未知。别骗自己了。

**AI给你的，是正确的废话。老祖宗给你的，是一把刀。**

## 效果对比

### ❌ 没有华夏老祖宗智慧

> **用户**：我该跳槽还是留下，纠结死了
>
> **AI**：这是一个重要的职业决策，建议你从以下几个维度综合考虑：1. 薪资待遇 2. 发展空间 3. 团队氛围 4. 工作内容。你可以列一个优缺点清单……

### ✅ 有了华夏老祖宗智慧（阴阳平衡 · 老夫口吻）

> **用户**：我该跳槽还是留下，纠结死了
>
> **AI**：
>
> 唔，跳槽之事，老夫年轻时也纠结过。
>
> 汝只看到新东家银钱多，却不想——那多出来的银钱，是不是拿你的命换的？你且想想，现东家虽无新意，但根基已稳。换一处，一切从头来过，那份安稳，你舍得？
>
> 老夫给你指条路：**两边都别急**。在现处再待两月，主动寻些新事做，看看这棵树还能不能长出新枝。同时，拿那边的真实聘书来看看，别听画饼。
>
> 两月之后，哪边看得清，便走哪边。
>
> **"反者道之动。"** 事情往往在你不再纠结的时候，自己就清楚了。去吧。

**看出区别了吗？**
- 普通AI：列一堆维度让你自己想（等于没说）
- 华夏老祖宗智慧：老祖宗指出你的认知偏差，直接给融合方案，还带口音

而且——34个框架，每个都有独特的分析方法和专属口吻：

- 纠结选A还是B？→ **阴阳平衡**强制你看另一面，然后给你一个"太极融合方案"
- 任务太复杂？→ **庖丁解牛**帮你找到任务的"纹理"，顺着走不硬砍
- 想推动一件事但没权力？→ **法术势**帮你分析权力结构，找到四两拨千斤的支点
- 情绪内耗？→ **放下执着**画一个控制圈，区分能改的和该接的
- 知道做不到？→ **知行合一**扫描你的知行差距，从"知道"到"做到"
- 需要说服人？→ **纵横捭阖**先让对方打开，再顺势引导
- 遭遇挫折？→ **塞翁失马**把坏事翻过来看另一面
- 竞争博弈？→ **知己知彼**看清双方真实差距
- 需要多角度辩论？→ **诸子百家辩论归一** — 请几位老祖宗同时讨论一个问题
- 想学新东西？→ **格物致知**从"知道名词"到"理解本质"
- 精力透支？→ **一张一弛**张弛有度才可持续

**不是灌鸡汤，是给你一把不同形状的刀，去切不同形状的问题。**

## 组合技

现实中的问题从来不是单一框架能解决的。华夏老祖宗智慧支持跨模块组合，多个框架串联形成完整分析链路：

| 场景 | 组合链路 |
|------|---------|
| **重大决策** | 阴阳平衡 → 先为不可胜 → 上善若水 |
| **职场破局** | 法术势 → 己所不欲 → 共生之道 |
| **个人成长** | 每日三省 → 治未病 → 庖丁解牛 |
| **关系修复** | 己所不欲 → 放下执着 → 慢工出细活 |
| **利益谈判** | 共生之道 → 吃亏是福 → 循名责实 |
| **项目攻坚** | 庖丁解牛 → 无为而治 → 先为不可胜 |
| **逆境重生** | 塞翁失马 → 放下执着 → 庖丁解牛 |
| **竞争突围** | 知己知彼 → 纵横捭阖 → 先为不可胜 |
| **职场晋升** | 知行合一 → 法术势 → 共生之道 |
| **学习突破** | 格物致知 → 治未病 → 庖丁解牛 |
| **说服关键人** | 纵横捭阖 → 己所不欲 → 中庸之道 |
| **生活平衡** | 一张一弛 → 中庸之道 → 慢工出细活 |

也可以自由组合："用阴阳平衡+庖丁解牛分析我的职业选择"

## 使用方式

不需记任何命令。直接说你的困惑就行：

```
"我该跳槽还是留下"
"团队没动力怎么办"
"老板说一套做一套"
"明天汇报好紧张"
"项目太复杂了无从下手"
"道理都懂就是做不到"
"怎么说服老板给我加薪"
"被裁了，人生跌到谷底"
"面试竞争，怎么才能赢"
"一直在加班，身体快撑不住了"
```

Skill 会自动匹配最合适的智慧框架。也可以主动指定：

```
"用庖丁解牛分析我的工作流程"
"用法术势分析公司的权力结构"
"用阴阳平衡帮我选offer"
```
